# MT-08 — Canary auto-rollback notification

## Mục tiêu
Verify user nhận notification realtime khi canary bị pause tự động do metric breach.

## Steps
1. Start canary 25%
2. Inject error spike (qua test endpoint)
3. Verify trong < 90s:
   - Toast notification "Canary paused — error rate breach"
   - Status badge chuyển sang "Paused"
   - Banner đỏ trong feature flag detail
4. Click toast → navigate vào detail

## Expected
- Notification trong 90s sau breach
- Toast persist cho đến user dismiss
- Sound/haptic optional cho mobile
