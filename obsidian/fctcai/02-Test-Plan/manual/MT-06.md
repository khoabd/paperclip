# MT-06 — RBAC kill switch hidden

## Mục tiêu
Verify non-admin user không thấy tab Kill Switch hoàn toàn (không chỉ disabled).

## Steps
1. Đăng nhập user role = "developer" (non-admin)
2. Mở workspace
3. Verify left sidebar/menu KHÔNG có item "Kill Switch"
4. Thử navigate trực tiếp `/workspace/:id/kill-switch` qua URL bar
5. Verify: redirect 403 hoặc home page

## Expected
- Tab hidden hoàn toàn
- URL navigate redirect/blocked
- API call `/api/kill-switch/...` trả 403

## Acceptance checklist
- [ ] Sidebar không có item Kill Switch
- [ ] URL direct navigate blocked
- [ ] API 403 cho non-admin
- [ ] Test với 3 role: viewer, developer, admin
