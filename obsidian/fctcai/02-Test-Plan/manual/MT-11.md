# MT-11 — Drag-in toggle UI

## Mục tiêu
Verify toggle "Tôi cần can thiệp tay (drag-in)" hiển thị trên mọi approval card và submit đúng vào metadata.dragIn.

## Steps
1. Mở Approval Center (web)
2. Click 1 approval card → drawer/modal mở
3. Verify toggle "I had to drag in" hiện ở footer
4. Default: off
5. Toggle on → submit approval
6. Verify approval_response.metadata.dragIn = true
7. Test trên mobile (responsive view) — toggle vẫn hiện

## Expected
- Toggle hiện trên mọi approval surface
- Default off
- State persist khi switch tab/page
- Mobile responsive

## Acceptance checklist
- [ ] Toggle hiện trên Approval drawer
- [ ] Default off
- [ ] Toggle state preserved khi navigate
- [ ] Submit lưu metadata.dragIn đúng
- [ ] Mobile: 320px width vẫn hiện
- [ ] Tooltip giải thích "drag-in" là gì
- [ ] a11y: keyboard navigable
