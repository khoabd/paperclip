# MT-09 — Magika file scan

## Mục tiêu
Verify khi user upload file (.doc, .pdf), Magika scan + inventory hiển thị trước khi proceed.

## Steps
1. Mở Greenfield intake form
2. Upload file `requirements.pdf` (~ 2MB)
3. Verify spinner "Scanning..." hiện ra
4. Sau scan: panel inventory hiển thị file type, size, content summary
5. Confirm proceed → file attached vào intake

## Expected
- Magika scan hoàn thành < 10s cho 2MB file
- Inventory panel có: filename, MIME type, size, summary
- Reject malicious file types (.exe, .sh) với clear error

## Acceptance checklist
- [ ] Spinner hiện trong scan
- [ ] Inventory panel populated
- [ ] Confirm gắn file vào intake
- [ ] Reject .exe upload

- [ ] File size limit (50MB?) enforce
