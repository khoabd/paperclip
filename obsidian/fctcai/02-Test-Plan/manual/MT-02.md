# MT-02 — Kill button state

## Mục tiêu
Verify kill button không clickable khi workspace idle (không có mission).

## Steps
1. Đăng nhập admin
2. Mở workspace mới (chưa có mission)
3. Mở Kill Switch panel
4. Verify nút "Freeze Workspace" disabled (mờ + cursor not-allowed)
5. Tạo 1 mission → nút enabled
6. Hoàn tất mission (status=done) → nút disabled trở lại

## Expected
- Button disabled khi 0 active missions
- Tooltip: "No active missions to freeze"

## Acceptance checklist
- [ ] Empty state: button disabled
- [ ] Active mission: button enabled
- [ ] After mission complete: button disabled
- [ ] Tooltip explains why disabled
- [ ] Visual: opacity hoặc grayscale rõ rệt
