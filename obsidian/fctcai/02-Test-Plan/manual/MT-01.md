# MT-01 — Double-confirm modal

## Mục tiêu
Tester verify modal yêu cầu gõ "CONFIRM" mới enable button khi action level ≥ 3.

## Steps
1. Đăng nhập admin user
2. Mở Kill Switch panel (workspace có active mission)
3. Click "Freeze Workspace" (level 3)
4. Verify modal hiện ra với:
   - Tiêu đề: "Confirm freeze"
   - Input field: "Type CONFIRM to proceed"
   - Button "OK" disabled
5. Gõ "confirm" (lowercase) → button vẫn disabled
6. Gõ "CONFIRM" (uppercase exact) → button enabled
7. Click cancel → modal đóng, không có action

## Expected
- Button chỉ enable khi text = "CONFIRM" exact
- Cancel không trigger action
- Modal có aria-modal=true (a11y)

## Acceptance checklist
- [ ] Modal xuất hiện với input
- [ ] Button disabled cho text != "CONFIRM"
- [ ] Button enabled cho "CONFIRM" exact
- [ ] Cancel button đóng modal
- [ ] Tab focus trapped trong modal
- [ ] Esc key đóng modal
- [ ] Screen reader announce modal mở/đóng
