# MT-03 — Empty title fallback

## Mục tiêu
Verify khi user submit intake không có title, hệ thống reject hoặc fallback "(untitled)".

## Steps
1. Mở Intake form
2. Để trống title field
3. Nhập text vào body
4. Click submit
5. Quan sát behavior

## Expected
- Option A: Form reject với inline error "Title required"
- Option B: Auto-fallback `title = "(untitled)" + first 30 chars body`
- Decision được document trong DEFAULT_DRAFTER spec

## Acceptance checklist
- [ ] Behavior consistent với spec (chọn 1 trong 2 option)
- [ ] Nếu reject: error message rõ ràng
- [ ] Nếu fallback: title preview hiện trước submit
- [ ] Sau submit: intake row có title đúng
