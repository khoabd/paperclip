# SM-07 — MCP health probe

## Mục tiêu
MCP health endpoint trả status hoặc circuit-open message.

## Steps
1. Configure GitLab MCP (mock)
2. Call `MCPHealthProbe.check('gitlab')`
3. Verify response: `{ status: 'healthy'|'broken'|'circuit_open' }`
4. Force MCP to fail → verify status='broken'

## Expected
- Healthy state recognized
- Failed state recognized
- Circuit open after threshold

## Acceptance checklist
- [ ] Healthy probe returns 'healthy'
