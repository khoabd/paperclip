# SM-09 — RBAC unauth

## Mục tiêu
GET `/workspaces/:id/health-score` trả 401 khi không có token.

## Steps
1. Call API không có Authorization header
2. Verify 401
3. Call với invalid token → 401
4. Call với valid token nhưng wrong workspace → 403

## Expected
- 401 cho missing/invalid token
- 403 cho wrong workspace

## Acceptance checklist
- [ ] No token: 401
- [ ] Invalid token: 401
- [ ] Wrong workspace: 403
- [ ] Valid: 200
